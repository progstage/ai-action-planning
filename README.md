## Opus Magnum AI Scheming
# Introduction
The goal of this project is to build an AI model that controls processes required to make deliberate schemes (i.e. plans focusing on successfully fulfilling given goals). The goals that can be planned for achieving are, as for their variety, somewhat limited. In short terms, just the kinds of goals that pertain to areas which we consider important are taken into account. We do not want to build an AI model helping make decisions on whether to buy this or that cone of ice cream, for instance.
